## Props

### Features1Widget

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description |
    |-----------|------------|------------|------------|
    | `columns` | `number` | `1` | Number of columns in the grid. From 1 to 4 |
    | `items` | `IItem[]` | `[]` | |
    | `title` | `string` | - | |
    | `subtitle` | `string` | - | |
    | `tagline` | `string` | - | |
    | `id` | `string` | - | |
    | `isDark` | `boolean` | `false` | |
    | `bg` | `string` | - | |
    | `classes.container` | `string` | - | |
    | `classes.headline.container` | `string` | - | |
    | `classes.headline.title` | `string` | - | |
    | `classes.headline.subtitle` | `string` | - | |
    | `classes.headline.tagline` | `string` | - | |
  </div>
</div>

### IItem

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description |
    |-----------|------------|------------|------------|
    | `as` | `string` | `article` | HTML tag name for the wrapper |
    | `asHeader` | `string` | `h3` | HTML tag name for the title |
    | `title` | `string` | - | |
    | `description` | `string` | - | |
    | `icon` | `string` | - | |
    | `callToAction` | `ICallToAction` | - | |
    | `data` | `INameValue[]` | - | |
    | `links` | `ILink[]` | - | |
    | `badge` | `string` | - | |
  </div>
</div>
